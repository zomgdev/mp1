Tool: select
Properties
Выберите entity, чтобы увидеть свойства.

Entity editor

Формат строк: name:type [meta]
Примеры meta: PK, FK, UQ, NN, IDX и т.п.
Пустые строки игнорируются.

Справка по редактору

  • Выбор/перемещение: инструмент Select (Hotkey: 1), тяните entity мышью.
  • Добавить entity: инструмент Entity (Hotkey: 2), клик по холсту.
  • Связь: инструмент Link (Hotkey: 3), два клика по разным entity.
  • Редактирование entity: двойной клик по заголовку.
  • Редактирование поля: двойной клик по строке поля.
  • Удалить связь: двойной клик по линии.
  • Панорамирование: удерживайте правую кнопку мыши и тяните.
  • Масштаб: колесо мыши над холстом.
  • Esc: отмена создания связи и закрытие редактора.
