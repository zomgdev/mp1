# Мегапроект

Создаю мегакомбайн для управления инфраструктурой во всех аспектах.
Включает в себя:
- управление серверами, виртуальными машинами, контейнерами, сетью, БД, безопасностью - их конфигурациями
- архитектурные инструменты, привязанные к реальности, а не хотелкам и фантазиям
- управление логикой, процессами, потоками данных
- управление проектами
- аналитика во всех аспектах со связями

Короче - универсальный инструмент для централизованного управлеинания инфраструктурой ИТ
В переспективе не только для ИТ.
